# 接口测试规范示例

## 通用校验

- 必填字段缺失时返回明确错误码。
- 用户未登录时返回未认证状态。
- 用户无权限时返回无权限状态。
- 同一请求重复提交时结果应幂等。
- 列表接口需要覆盖分页边界。

## 报名接口

请求字段:

- course_id: 课程 ID。
- class_id: 班级 ID。
- user_id: 用户 ID。

重点场景:

- 未登录报名。
- 已报名重复提交。
- 班级满员。
- 课程下架。
- 支付订单创建失败。

## 支付回调接口

重点场景:

- 支付成功。
- 支付失败。
- 回调重复通知。
- 订单已关闭后回调。
- 签名错误。
