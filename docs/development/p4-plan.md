# P4 测试点确认与用例模板开发计划

## 目标

满足 PRD 中两项核心要求:

- 用户确认或编辑测试点。
- 按用户模板生成测试用例。

## 交付范围

1. 结构化测试点表格。
2. 测试点编辑后同步回 `ApplicationService.current_points`。
3. 编辑测试点后重新生成用例使用最新测试点。
4. `CaseTemplate` 表示用户选择的用例字段和字段顺序。
5. Markdown / Excel 导出按模板字段输出。
6. 模型生成 Prompt 包含模板字段约束。

## 不进入范围

- 模板市场。
- 多模板管理。
- 复杂字段类型校验。
- 表格拖拽排序。

## 验证方式

- 服务层测试覆盖测试点同步、模板导出、Prompt 模板字段约束。
- 全量单元测试通过。
- UI 编译检查通过。
